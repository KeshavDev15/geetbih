# Design Specification: GEETBIH.COM (2026)

## 1. Vision & Persona
- **Persona:** "AI Factory inside a Minimalist Digital Studio."
- **Objective:** Showcase high-end engineering, AI automation, and branding capability to business clients.
- **Tone:** Innovative, Reliable, High-Quality.

## 2. Visual Identity (Obsidian Pulse)
### Color Palette (OKLCH)
- **Background (Obsidian):** `oklch(15% 0.01 260)` - Deep slate-black for high-trust depth.
- **Surface (Carbon):** `oklch(22% 0.01 260)` - For cards and sections; creates elevation without borders.
- **Accent (Electric Mint):** `oklch(85% 0.16 170)` - High-energy green representing the AI layer and velocity.
- **Text (Cloud Dancer):** `oklch(97% 0.01 260)` - Off-white for clarity and ocular comfort.
- **Muted (Silver):** `oklch(65% 0.01 260)` - Technical metadata and labels.

### Typography
- **Headings:** High-Contrast Serif (e.g., *Editorial New*) - Represents Branding and Quality.
- **Body:** Geometric Sans-Serif (e.g., *Geist*) - Represents Engineering and Reliability.
- **Technical/AI:** Precision Mono (e.g., *IBM Plex Mono*) - Represents the "AI Factory" logic.

## 3. Experience & Motion
- **Scroll:** Lenis smooth scrolling for a premium feel.
- **Interaction:** GSAP "Staggered Assembly" (elements precisely building in as you scroll).
- **AI Visuals:** Subtle Three.js interactive "Data Node" field in the background.

## 4. Key Sections (Home)
1. **Hero:** "GEETBIH LABS" assembly animation with abstract 3D core.
2. **Capability Layers:**
   - Visual Branding (Large imagery)
   - Engineering (Structural/Code aesthetics)
   - AI Automation (Flowing data nodes)
3. **Product Orbit:** Immersive horizontal portfolio gallery.
4. **The Audit Engine:** High-end technical form for client lead generation.

## 5. Technical Stack
- **Frontend:** Next.js 16, Tailwind v4, GSAP, Framer Motion, Lenis, Three.js.
- **Backend:** Nest.js, Prisma 7, PostgreSQL.
- **Deployment:** Vercel (Web), DigitalOcean (API/DB).
